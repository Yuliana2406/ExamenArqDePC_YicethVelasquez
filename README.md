# Examen Final Arquitectura de Computadores

## Objetivo
Desplegar una infraestructura en AWS utilizando CloudFormation y GitHub Actions.

## Tecnologías
- AWS EC2
- CloudFormation
- GitHub Actions
- Git

## Deploy
Crea una instancia EC2 con acceso HTTP y una página web.

## Destroy
Elimina toda la infraestructura creada.

## Template
El template crea una instancia EC2 Amazon Linux y un Security Group con acceso al puerto 80.
